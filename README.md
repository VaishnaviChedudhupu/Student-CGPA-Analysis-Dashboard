# 🎓 Student CGPA Analysis Dashboard | Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard built to analyze student academic performance using CGPA data.

The dashboard provides insights into student performance through KPIs, interactive visualizations, DAX measures, and filters. It helps identify top-performing students, average performance, and students who require academic improvement.

---

# 📊 Dashboard Pages

## 1. Executive Dashboard

A high-level overview of student academic performance.

KPIs Included:

- Total Students
- Average CGPA
- Highest CGPA
- Lowest CGPA

Visuals:

- Top 10 Students by CGPA
- CGPA Category Distribution
- Student Performance Table
- Interactive Student Filter

---

## 2. Above Average Students

Displays students whose CGPA is greater than or equal to the overall average.

Insights:

- Above Average Student List
- Individual CGPA
- Student Ranking
- Performance Summary

---

## 3. Below Average Students

Displays students whose CGPA is below the overall average.

Insights:

- Below Average Student List
- Individual CGPA
- Performance Analysis
- Academic Improvement Candidates

---

## 4. Student Data View

Detailed student information including:

- Student Name
- Hall Ticket Number
- Father Name
- Final CGPA
- CGPA Category
- Above/Below Average Status
- Rank
